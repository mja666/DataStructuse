DataStructure 项目 — Python 文件功能一览
（生成时以仓库内 .py 为准）

--------------------------------------------------------------------------------
fleet_simulation.py
--------------------------------------------------------------------------------
新能源车队离散事件仿真主程序与核心模型：网格道路图（四邻接、障碍格）、
Dijkstra 最短路（按源点缓存）、多车/电量/载重、任务按时间随机到达与截止、
充电站排队与槽位、得分（早到奖励/迟到与过期罚/路程罚）。策略为仓库重量
贪心多票装车 + 最近邻配送顺序 + 送完一批再回仓。含 SMALL/MEDIUM/LARGE 等
预设场景、summarize 统计、可选任务 CSV 导出。

--------------------------------------------------------------------------------
fleet_nearest_first.py
--------------------------------------------------------------------------------
与主仿真并列的策略变体：继承 FleetSimulator，仅将「仓库装批」改为按图最短路
距离的最近邻贪心装车；其余行驶、充电、回仓、得分与可视化接口仍复用基类。
可直接运行打印各预设场景下的 summarize。

--------------------------------------------------------------------------------
fleet_metaheuristic.py
--------------------------------------------------------------------------------
控制变量对比实验：MetaHeuristicFleetSimulator（同重 EDD 平局装批 + 批内 SA/
小规模全排列优化访问序）、MetaHeuristicNearestFleetSimulator（最近装批 + 同上
路线优化）。命令行可跑重量线/最近线对比、--all、--visual 打开含多策略的
Tk 可视化（依赖 fleet_visual）。

--------------------------------------------------------------------------------
fleet_rl_max_weight.py
--------------------------------------------------------------------------------
强化学习（REINFORCE）在「重量贪心装批」前提下学习同一批内的配送顺序；
与仿真中代理代价对齐。支持训练与对比实验（见文件内 argparse 说明）。
依赖：标准库 + fleet_simulation。

--------------------------------------------------------------------------------
fleet_visual.py
--------------------------------------------------------------------------------
网格车队仿真的 Tk 图形界面：场景与策略下拉、播放/暂停/重启，绘制格子路网、
障碍、仓库、充电站、任务、规划路径、轨迹尾迹、车辆与电量条、进度与得分。
沿最短路按距离插值显示车辆位置；供 fleet_metaheuristic --visual 等复用。

--------------------------------------------------------------------------------
fleet_osm.py
--------------------------------------------------------------------------------
将同一套车队模型（多车、电量、充电、动态任务、得分）搬到 OSM 折线构图
（Haversine 米为边权）上：FleetSimulatorRoad 继承 FleetSimulator；Tk Canvas
按经纬度线性投影绘制路网与动态演示。支持联网拉 Overpass 或 --csv 离线路网；
--no-gui 仅跑仿真打印 summarize。

--------------------------------------------------------------------------------
osm_graph.py
--------------------------------------------------------------------------------
OSM 路网共用模块：路段数据结构 Segment、Haversine 距离、Overpass 查询（车行
highway 正则过滤）、JSON 解析为折线段、CSV 读写、RoadGraph（量化顶点无向图）。
供 fleet_osm 等引用；无独立命令行入口。

--------------------------------------------------------------------------------
osm_fetch_demo.py
--------------------------------------------------------------------------------
试验脚本：仅用标准库向 Overpass 请求小 bbox 内 highway 路网（out geom），
统计并写出 osm_sample_segments.csv。用于快速拉数或离线给 fleet_osm --csv 使用。
